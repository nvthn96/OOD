Class không được chứa hàm mà nó không có.

Có interface là IPrinter với methods: Print, Fax, Scan

2 class là
class HPrinter : IPrinter {
	Print() {}
	Fax() {}
	Scan() {}
}

class JetPrinter : IPrinter {
	Print() {}
	Fax() {}
	Scan() { throw exception; }
}

Như vậy interface IPrinter đã vi phạm nguyên tắc vì nó quá nhiều method mà class con không thực hiện được.
Cần phải chia nhỏ interface thành
interface IPrinterTask: Print, Fax
interface IPrinterExtend: Scan

Khi đó, 2 class sẽ là
class HPrinter : IPrinterTask, IPrinterExtend {
	Print() {}
	Fax() {}
	Scan() {}
}

class JetPrinter : IPrinterTask {
	Print() {}
	Fax() {}
}
